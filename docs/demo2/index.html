auvitex example

Auvitex is a JavaScript library for a podcast player, that plays audio and/or video podcasts with synchron text. The text is displayed just like video subtitles. It's easy to navigate the text. You can use the cursor keys of your keyboard too!

Audio content is a LibriVox recording. Text is from gutenberg.org.
